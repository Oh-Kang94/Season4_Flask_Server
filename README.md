### 간략 설명 
1. JWT를 이용한 인증 방식의 보안 구성
2. SwaggerUI를 통한, Fromtend과의 협업
3. Folium을 이용한, 지도 구현
4. Tensorflow,Kiwipie를 이용한 한국어 tokenizer 후, 추천 영화 리뷰점수 예측.

### 설치 방법
    pip install -r requirements.txt
    ** 실행시,
    flask run
    ** 백그라운드에서 실행시,
    nohup flask run --host=0.0.0.0
